About Tree of Science

Search less, research more!

Tree of Science (ToS) is a Web based tool that uses sophisticated network algorithms to optimize the search and selection of published articles. It is an especially useful tool for those interested in tracking how a particular topic or paper is interconnected over time with others. ToS is likely to be useful for both dedicated researchers and students working on short-term research projects.
